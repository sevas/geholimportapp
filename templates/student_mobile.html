Hello. You probably went here by picturing a QR code. This dedicated page will allow you to subscribe to calendars on your mobile device, just by opening the links.


As far as we know, this only works with an iPhone, an iPad or an iPod touch.

Attention,
For some reason, it appears the latest version of iOS (1.4.x) does not allow to add a calendar by following an .ics link.
This is very unfortunate and we're not really sure of the reason Apple did that.


It is still possible to add an iCalendar subscription through the calendar settings of your iOS device. Proceed as following:

  • Copy the ics link in your clipboard
  • Go to Settings
  • Go to Mail, Contacts, Calendar
  • Select "Add Account"
  • Select "Other"
  • In the "Calendars" section, select "Add Subscribed Calendar"
  • Paste the link in the box


Yes, this is not as nice as before. As the older behavior is still documented in the official documentation, we hope the next iOS release will fix back what wasn't broken.
