About

GeHoL is the scheduling system used at Brussels University. It is based on the Scientia® Course Planner platform. We think its interface is abysmal. We think no one should be forced to use what can best be described as a slightly-below-average amateur project made in the mid-nineties.

Gehol Importer is a tool to convert GeHoL queries into something usable by calendar software such as Google Calendar, Apple iCal, Mozilla Sunbird, Microsoft Outlook, the way it should have been from the start.

This service is still under development. We will try to keep the core functionality running, but you are likely to experience changes as we add features and improve usability. For more information, consult this page.

The authors of this application are not affiliated with either the original GeHoL service or Scientia®. Questions, comments and suggestions should be sent to the following address : geholimportapp@gmail.com
